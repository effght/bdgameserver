# Badminton FastAPI Server

This is a standalone FastAPI wrapper for the badminton simulator.

It serves:

- `GET /`: English landing page.
- `GET /player`: player-facing simulator page.
- `GET /hit`: phone controller page.
- `WS /ws`: WebSocket signaling bridge with room/channel routing.
- `POST /api/rooms`: create a UUID room id.
- `POST /api/stats`: placeholder for future match stats.
- `GET /api/users/me`: placeholder for future auth.

## Local Run

```bash
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8765 --reload
```

Open:

```text
http://localhost:8765/
```

Use `/player` to open the court. The QR code generated by the player page points the phone controller to the same host and `/ws`.

## Render

Use `fastapi_app` as the Render root directory.

Build command:

```bash
pip install -r requirements.txt
```

Start command:

```bash
uvicorn app.main:app --host 0.0.0.0 --port $PORT
```
